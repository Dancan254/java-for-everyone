# Introduction to Java

This section covers the foundational concepts every Java program is built on. Work through the topics in order — each one builds on the previous.

## Learning Objectives

By the end of this section, you should be able to:

- Understand and distinguish between Java's primitive data types
- Declare, initialize, and use variables effectively
- Apply proper naming conventions for variables
- Use arithmetic and assignment operators to perform calculations
- Understand type casting and conversion between numeric types
- Read user input using the Scanner class
- Write programs that process and manipulate String data
